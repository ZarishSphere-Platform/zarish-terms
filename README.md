# Zarish-Terms (Terminology Management)

Terminology and code system management for ZarishSphere Platform.

## Models

- **CodeSystem**: SNOMED CT, ICD-11, LOINC, etc.
- **Concept**: Individual codes
- **ValueSet**: Sets of codes

## Technology Stack

- Go 1.21+
- GORM ORM
- PostgreSQL

## Port

8086 (planned)
